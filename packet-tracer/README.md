# Packet Tracer File

Place the final Cisco Packet Tracer `.pkt` file in this folder.

Before publishing it, verify:

- The topology matches the `10.0.0.0/20` plan
- All VLAN IDs and gateway addresses match the documentation
- RIPv2 routes are learned successfully
- Port-security modes behave as documented
- No real passwords or personal information remain
- The file opens correctly in the intended Packet Tracer version
